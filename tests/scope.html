Scope - JavaScript

This examples shows that scope depends on where things are defined.

In this example the function funcB() is defined in global scope, so when calling it in function funcA(), the outer scope for funcB() is the global scope and not funcA()!

Function funcC(), on the other hand, is defined within function funcA(), so outer scope of funcC() is scope of function funcA().
